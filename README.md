# Smart Notes App 🧠

A full-stack notes application with authentication and AI support.

---

## 🚀 Features

- User Signup/Login (JWT)
- Create, Edit, Delete Notes
- Search Notes
- Pin Important Notes
- Timestamp support
- AI-powered note improvement (mock AI)

---

## 🛠 Tech Stack

- Next.js
- MongoDB
- Node.js
- Tailwind CSS
